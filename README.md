# val

Easily store and retrieve key/value pairs in `~/.key-valerie`

## USAGE

### Setting a token:

    $ val foo bar

### Getting a token..

    $ val foo

...will return "bar"

### Unsetting a token...

    $ val foo --unset

...will delete the "bar"

### Listing all tokens:

    $ val

That's it and that's all folks.
